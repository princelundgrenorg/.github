# princelundgrenorg

Small, focused macOS utility apps, built and shipped independently
through a shared CI/release pipeline.

## Common ground

Apps ship either through the Mac App Store or via Developer ID +
Sparkle updates, built and signed through this org's shared
release workflows. All follow this org's
[security policy](https://github.com/princelundgrenorg/.github/blob/main/SECURITY.md)
and
[code of conduct](https://github.com/princelundgrenorg/.github/blob/main/CODE_OF_CONDUCT.md).

## How to report things

- **Bug in a specific app** → file an issue on that app's repo
- **Security vulnerability** → see
  [SECURITY.md](https://github.com/princelundgrenorg/.github/blob/main/SECURITY.md)
  (private channel, don't post publicly)
